# GENTEX-Valuation-Models
An institutional-grade corporate valuation and 10-year financial forecasting model for Gentex Corporation. This project analyzes historical financial performance from 2021 to 2025 and builds a multi-scenario projection runway (2026–2030) to evaluate the company's intrinsic value in support of a buy case of this company.

The model specifically accounts for the operational and structural impacts of the 2025 VOXX International acquisition, addressing segment reporting revisions, integration synergies, and macroeconomic automotive production trends.
